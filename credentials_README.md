# credentials.json について

このファイルには各治験サイトのログイン情報が含まれます。

## 重要な注意事項

- **他人と共有しない** — クラウド同期フォルダに置く場合は注意
- **Gitなどのバージョン管理に含めない** — もし将来リポジトリ化する場合
- **チャットで直接共有しない** — 会話履歴に残ります

## 編集方法

1. テキストエディタ（メモ帳、VSCode、等）で `credentials.json` を開く
2. 各サイトの `email` と `password` を入力して保存
3. scraper.py が次回実行時に自動読み込み

## フィールド

- `email`: ログイン用メールアドレス（またはユーザー名）
- `password`: パスワード
- `login_url`: ログインフォームのURL（空でもOK、自動検出）
- `site_name`: 参考表示用、変更不可

## 使わないサイトは

`email` と `password` を空のままにしておけば、非ログイン状態でスクレイピングします（金額情報取得不可のサイトもあり）。
